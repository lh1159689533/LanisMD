## ADR 示例

### ADR-001: 选择 React 作为前端框架

**状态**: 已接受
**日期**: 2026-02-02

**背景**：
我们需要为新的 Web 应用选择前端框架。团队有 3 名前端开发人员，其中 2 人熟悉 React，1 人熟悉 Vue。项目需要在 3 个月内完成 MVP。

**决策**：
我们决定使用 React 18 + TypeScript 作为前端技术栈。

**理由**：
1. 团队大多数成员熟悉 React
2. React 生态系统成熟，组件库丰富
3. TypeScript 支持好，有利于代码质量
4. 社区活跃，问题容易找到解决方案

**后果**：
- ✅ 可以快速上手开发
- ✅ 丰富的第三方组件可用
- ⚠️ 一名 Vue 开发者需要学习 React
